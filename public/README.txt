תיקייה זו מכילה נכסים סטטיים שמועברים כפי שהם בעת ה-build.

נכסים נדרשים (יש להעלות בפועל):
  logo.png           - לוגו האפליקציה (יוצג ב-Header). מומלץ ~200x200 px
  icon-192.png       - אייקון PWA 192x192
  icon-512.png       - אייקון PWA 512x512
  apple-touch-icon.png - אייקון iOS (180x180)

הקבצים האלה ייווצרו אוטומטית מהלוגו שהמשתמש יעלה.
עד אז: vite-plugin-pwa יחפש את הקבצים בזמן build ויפלוט אזהרה אם חסרים -
התקנת הקבצים מבטיחה התנהגות PWA תקינה במובייל.
